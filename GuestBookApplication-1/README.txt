# Steps to test GuestBook Application

-> execute the sql file in your mysql env to create db, tables, data
-> import to your Eclipse/STS ide
-> modify application.properties file according to your mysql db credentials.
-> perform Maven clean install
-> open the java com.techm.controller.GuestBookApplication1 right click on the file->Run as->Spring boot App.
-> link to access the guest book application is http://localhost:8080/index.jsp
-> Application screenshots are available in "Guestbook Screens.Doc" file
